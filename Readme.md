# 📕 프로젝트 소개
- 저희 YAHO는 등산을 좋아하시는 분들이 국내의 산 정보를 찾아 보실 수 있게 만든 프로그램입니다.
- 메인 컨텐츠에서는 원하는 지역과 산을 선택해 산에 대한 정보를 찾아 보실 수 있습니다.
- 메인 컨텐츠 이용 후 만족도와 관련해 리뷰 작성이 가능합니다.
